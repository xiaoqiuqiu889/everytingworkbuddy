# EFW — Everything For WorkBuddy / CodeBuddy

> 🍶 **一杯给 AI 助手的翡翠剂。**
> 干了这杯，WorkBuddy / CodeBuddy 从「顺手的工具」觉醒成「身经百战的战术搭档」——懂你是谁、会自己挑装备、上阵不掉链子。

对标 [Everything Claude Code](https://github.com/worldflowai/everything-claude-code)，用 WorkBuddy 原生机制重造的**开箱即用能力全家桶**。不用懂配置，灌下去，说人话就能用。

---

## 🍶 两口灌下去

**第一口 · 注入** —— 把仓库链接甩给你的 WorkBuddy（或 CodeBuddy），说一句：

> 🔗 https://github.com/xiaoqiuqiu889/everytingworkbuddy
> 「把这个工程在我的电脑中实现一遍」

它自动 **克隆 → 安装 → 自检**，全套战力一次装满。
（CodeBuddy 用户：让它跑 `node scripts/install.mjs --product codebuddy`，装进 `~/.codebuddy/`。）

**第二口 · 觉醒** —— 开个新会话，报上你的身份：

> 「我是一名 **游戏策划**，平时 **写设计文档、做数值系统和竞品拆解**，让 EFW 帮我升级下 workbuddy。」

EFW 当场 **检索最趁手的能力 → 排好优先级 → 刻进你的档案**。往后每次开工，助手都记得你是谁、该抄哪把家伙。

**收尾 · 通电（手动一步）** —— 两个增强连接器（`context7` 查最新文档、`sequential-thinking` 深度推理）出于安全机制需你亲手激活：

> 左侧边栏 →「连接器」→ 找到 `context7` 和 `sequential-thinking` → 各点一次「信任」（无需密钥）。

> CodeBuddy 用户免此步，装完自动生效。想要「专家中心」一键拉起的专家？再说句「把子代理转成专家」即可。

## 灌下去，强在哪

一杯翡翠剂配齐三样东西：**一套战术纪律、一座能力军械库、一个懂你的参谋。**

- **参谋（能力检索）**：不用背技能名。一句「我是做 XX 的」，它就从军械库里挑出最该上手的组合，刻进档案，此后每次对话自动带上。
- **军械库（306 条能力索引）**：按 **生态真实使用热度 × 多角色覆盖** 编排，办公 / 研发 / 数据 / 设计 / 营销 / 金融 / 垂直行业全覆盖——连你本机还没装的高热度能力也照样帮你检索到。
  > 检索结果分两区：**✅ 现在就能用**（说句需求就触发）和 **📦 装了/授权就能用**（需先装技能、点开专家或信任连接器）。能检索到 ≠ 已到手，EFW 会告诉你每样怎么上膛。
- **越用越强**：往后装的任何新技能，都自动并入检索。军械库随你成长，不封顶。

## 药效（提升了什么）

| 提升 | 具体表现 |
| --- | --- |
| **少走弯路** | 先定位报错根因、先写测试再实现；需求跑偏在第一步就被拦下 |
| **出手更稳** | 固定清单查漏洞 / 密钥硬编码 / 边界条件，不靠临场发挥 |
| **情报更准** | 连最新官方文档，不拿过时知识瞎编 |
| **上阵更快** | 工作规范随每个新会话自动加载，新人 / 新项目不用口口相传 |
| **防走火** | 专抓密钥硬编码、`.env` 进库、路径穿越等高危操作 |
| **懂你** | 报一句身份就自动配好装备，省去反复交代「我是做啥的」 |
