# EFW — Everything For WorkBuddy / CodeBuddy

> 🍶 **一杯给 AI 助手的翡翠剂。**
> 干了这杯，WorkBuddy / CodeBuddy 从「顺手的工具」觉醒成「身经百战的战术搭档」——懂你是谁、会自己挑装备、上阵不掉链子。

EFW（Everything For WorkBuddy）是一套开箱即用的 WorkBuddy / CodeBuddy 能力配置包，对标 [Everything Claude Code](https://github.com/worldflowai/everything-claude-code)。它把技能、专家、子代理、连接器、准则、工作流编排成一套带「编排外壳」的即用能力全家桶，无需手动写配置，用自然语言即可调用。

---

## 安装

1. **落地仓库** —— 把本仓库链接甩给你的 WorkBuddy（或 CodeBuddy），指令：
   > 「把这个工程在我的电脑中实现一遍」
   >
   > 🔗 https://github.com/xiaoqiuqiu889/everytingworkbuddy

2. **写入身份档案** —— 开个新会话，报上你的角色与日常任务，例如：
   > 「我是一名游戏策划，平时写设计文档、做数值系统和竞品拆解，让 EFW 帮我升级 WorkBuddy。」
   EFW 会检索最匹配的能力组合并写入长期档案，此后每次会话自动携带。

3. **激活增强连接器（可选）** —— `context7`（查最新官方文档）与 `sequential-thinking`（深度推理）出于安全机制需手动信任：左侧边栏 →「连接器」→ 找到两项各点一次「信任」（无需密钥）。CodeBuddy 用户免此步，装完自动生效。

## 核心组成

EFW 由三层构成：**能力检索（参谋）** + **能力索引** + **身份档案（懂你）**。

- **能力检索**：无需记忆技能名。一句「我是做 XX 的」即触发检索，挑出最该上手的能力组合写入档案，后续会话自动带上。
- **能力索引（306 条）**：按生态真实使用热度 × 多角色覆盖编排，办公 / 研发 / 数据 / 设计 / 营销 / 金融 / 垂直行业全覆盖。检索结果分两区——**✅ 现在就能用**（说句需求即触发）与 **📦 装了/授权就能用**（需先装技能、开专家或信任连接器）。
- **身份档案**：一次写入，后续所有会话自动携带，免去反复交代角色。
- **可成长**：后续安装的任何新技能自动并入检索索引，不封顶。

## 编排外壳（四步）

EFW 的核心是一套「编排外壳」，对**任意实质性任务**统一走四步：

1. **先想再做（计划）** —— 拆解目标、交付物形态、分步与验收标准。
2. **干活** —— 交给对口领域技能或原生工具执行。
3. **交付前验收** —— 对照验收标准逐条勾选，产出可核验清单（文档 / 数据 / 设计任务同样强制，不豁免）。
4. **做完沉淀** —— 可复用套路写入长期记忆或存为新技能。

> **验收闸门**：未跑验收之前，任务不算「完成」。文档 / 数据 / 设计类任务最易被当成「领域技能任务」而跳过，须强制走闸门。

## 安全约束

- 禁止硬编码密钥 / token / 密码，一律走环境变量；`.env` 不进库。
- 外部输入零信任（防注入 / XSS / 路径穿越）。
- 专抓密钥硬编码、`.env` 进库、路径穿越等高危操作。
