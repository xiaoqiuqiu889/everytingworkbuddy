# EFW — Everything For WorkBuddy / CodeBuddy

> 🍶 **为 WorkBuddy 准备的翡翠剂** —— 干了这杯翡翠剂，WorkBuddy 更强大。
>
> *（逆战世界观彩蛋：翡翠剂是逆行者出征前吞下的强化剂；EFW 就是给 WorkBuddy 的这一剂。）*

> 对标 [Everything Claude Code](https://github.com/worldflowai/everything-claude-code)（Claude Code 的配置全家桶），为 **WorkBuddy / CodeBuddy** 重新打造的「开箱即用的全能工作助手配置包」。

EFW 把一套经过实战打磨的工作流、能力索引和使用纪律，用 WorkBuddy / CodeBuddy 原生机制一次性装好。你不用懂任何配置，装完直接说人话就能用。

> **同时兼容 CodeBuddy**：CodeBuddy 与 WorkBuddy 共用同一套技能（`skills/`）和 MCP（`mcp.json`）格式。EFW 安装器支持 `--product codebuddy`，一键把全部能力装到 `~/.codebuddy/`；能力检索也会自动识别两个产品已装的技能。

---

## 👋 怎么用（两步）

### ① 首次安装：把链接发给 AI，让它自己装

把下面这个仓库链接直接发给你的 WorkBuddy（或 CodeBuddy），说一句：

> 🔗 https://github.com/xiaoqiuqiu889/everytingworkbuddy
> 「把这个工程在我的电脑中实现一遍」

它会自动 **克隆仓库 → 运行安装 → 自检通过**，一次性装好全部能力。

- **CodeBuddy 用户**：发给 CodeBuddy 时，安装器默认装到 WorkBuddy；请让它运行 `node scripts/install.mjs --product codebuddy`，即可装到 `~/.codebuddy/`。

### ② 自定义升级：告诉 AI 你是谁、做什么

装完开个新会话，直接说（换成你的真实身份）：

> 「我是一名 **游戏策划**，我平时会 **写设计文档、做数值系统和竞品拆解**，让 EFW 帮我升级下我的 workbuddy。」

EFW 会 **自动检索最适合你的能力组合** → 排好优先级 → 写进你的个人档案。之后 **每个新会话** 你的助手（WorkBuddy / CodeBuddy）都自动知道你是谁、该优先用哪些能力。

### 装完还需手动 1 步

EFW 内置了 2 个增强连接器（`context7` 查最新文档、`sequential-thinking` 深度推理），但 WorkBuddy 出于安全机制要求你手动点一次「信任」才生效：

> 左侧边栏 → 点击 **「连接器」** → 找到 `context7` 和 `sequential-thinking` → 分别点 **「信任」**（无需填密钥）

> 💡 **CodeBuddy 用户无需此步**：在 CodeBuddy 里这 2 个连接器安装后自动生效，不必手动信任。

> 想要「专家中心」里可一键拉起的专家？再说一句「把子代理转成专家」即可（可选，进阶玩法）。

## 用来做什么

EFW 给 WorkBuddy 补齐一套「**懂你、会挑能力**」的系统：

- **能力检索（Profile）**：不用背技能名、不用读文档。说一句「我是做 XX 的、平时做 YY」，它就从 **300 级条覆盖多角色的能力库**里挑出最该用的组合（技能 / 文档 / 数据 / 设计 / 办公 / 多模态…），自动写进你的个人档案，以后每次对话都自动带上。
- **一份随大流的能力库**：这 300 级条不是拍脑袋定的，而是按 **WorkBuddy 生态里广大用户真实使用热度 + 多角色覆盖**编排的——办公协作、开发、数据分析、创意设计、营销、金融、垂直行业等使用面最广的能力全在里面，连你本地没装的生态高热度能力（如批量文件处理、OCR+知识库、数据分析+PPT）也会被检索到。
- **能力会自己长大**：你以后在 WorkBuddy 里装的任意新技能，都会被自动纳入检索，索引随你使用自增长，不会卡死。

## 这 300+ 能力都分哪些类

EFW 的能力索引不是一堆散装技能，而是按 **「机制类型 × 领域角色」** 两层组织，共 **306 条**：

**按机制类型（WorkBuddy 原生能力载体）**
| 类型 | 数量 | 干什么用 |
| --- | --- | --- |
| 技能 Skill | 161 | 直接干活的能力模块（读写 / 生成 / 分析 / 设计…） |
| 专家 Expert | 49 | 领域顾问，一句话拉起深度咨询 |
| 子代理 Agent | 40 | 专职角色，并行啃复杂任务 |
| MCP 连接器 | 21 | 打通外部系统（数据库 / CRM / 设计 / 搜索…） |
| 准则 Rule | 20 | 每次会话自动生效的行为红线 |
| 工作流 Workflow | 15 | 多步骤自动化流水线 |

**按领域角色（你大概率用得上的）**
- 📄 **办公协作**：PDF / Word / Excel / PPT / 腾讯文档 / 邮件 / 会议 / 项目管理（Jira、Trello）/ 知识库
- 💻 **研发工程**：编码规范 / TDD / 代码评审 / 构建修复 / 安全审计 / 部署 / 容器 / 数据库
- 📊 **数据分析**：清洗 / 可视化大屏 / 报表 / 用户行为 / A-B 测试 / 增长
- 🎨 **创意设计**：PPT 母版 / 图像 / 视频 / UI 原型 / 品牌 / 插画 / 信息图 / 动效
- ✍️ **内容营销**：写作 / SEO / 小红书 / 抖音 / 公众号 / 广告 / 落地页 / 邮件营销
- 💹 **金融投资**：股票 / 基金 / 量化 / 财报 / 风控 / 税务
- 🏥 **垂直行业**：医疗 / 教育 / 电商 / 法务 / 房产 / 政务 / 客服 / 供应链
- 🔌 **自动化与连接器**：定时任务 / 跨系统同步 / 多专家并行 / 外部 MCP 集成

## 提升了什么

装了 EFW，WorkBuddy 从「你每次都要手把手教它」变成「它知道你是谁、该走什么流程」：

| 提升点 | 具体表现 |
| --- | --- |
| **少走弯路** | 写代码会强制先定位报错根因、先写测试再实现；需求跑偏在第一步被拦住 |
| **审查更稳** | 固定清单查安全漏洞 / 密钥硬编码 / 边界条件，不再靠临场发挥 |
| **查资料更准** | 连上最新官方文档，不拿过时知识瞎编 |
| **上手更快** | 工作规范自动随每个新会话加载，新人 / 新项目不用口口相传 |
| **防事故** | 专抓密钥硬编码、`.env` 进库、路径穿越等高危操作 |
| **懂你** | 说一句身份就自动匹配能力，省去反复交代「我是做啥的」 |

## 对标什么产品

EFW 的灵感与设计目标，来自给 **Claude Code** 用的爆款配置包 [Everything Claude Code](https://github.com/worldflowai/everything-claude-code)（含 agents / skills / hooks / commands / rules / mcp 六类能力）。

但 **Claude Code 和 WorkBuddy 是两套不同产品，配置体系不能直接照搬**。EFW 做的是：把那套包里每一类能力的 **价值**，用 WorkBuddy 自己的原生机制重新实现一遍，让你在 WorkBuddy 里也能获得同等「一装就专业」的体验——并且更进一步，加了 **按用户身份自动检索能力** 的开箱即用机制。
